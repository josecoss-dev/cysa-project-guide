Portfolio projects

8 projects. All 4 exam domains.

Each project maps directly to CS0-003 exam objectives. Build them, push to GitHub, and walk into interviews with working code.

Security ops (33%) Vuln mgmt (30%) Incident response (20%) Reporting (17%)
01 / 08
Wazuh SIEM home lab
Deploy a full open-source SIEM/XDR stack, write custom detection rules, simulate attacks, and document alert triage. Industry-grade — real SOCs run Wazuh.
Security ops Incident response Intermediate 2–3 weeks
02 / 08
Vuln scan & remediation pipeline
Nessus Essentials (free) against Metasploitable VMs. Prioritize CVEs by CVSS, write remediation reports, and automate triage with Python.
Vuln mgmt Beginner 1–2 weeks
03 / 08
Phishing triage CLI tool
Python CLI that parses email headers, extracts IOCs, queries the VirusTotal API, and outputs a structured analyst report. Ships to GitHub day one.
Security ops Incident response Beginner 1 week
04 / 08
Threat hunt: BOTS v3 dataset
Load a real APT attack dataset into Splunk and hunt adversary activity across kill chain stages. Write hunt reports exactly like a SOC analyst. Employers love this one.
Security ops Intermediate 2–3 weeks
05 / 08
Malware traffic analysis (pcap)
Analyze real malware pcap files from malware-traffic-analysis.net. Extract IOCs with Wireshark + Zeek, map to ATT&CK, write weekly analyst reports.
Security ops Vuln mgmt Beginner Ongoing
06 / 08
Forensic investigation lab
Memory and disk forensics with Volatility 3 and Autopsy. Reconstruct attacker timelines on compromised images. Write formal forensic reports. Most candidates skip this — don't.
Incident response Advanced 3–4 weeks
07 / 08
TryHackMe SOC Level 1 path
Guided, browser-based rooms covering every CySA+ domain — SIEM, threat hunting, IR, phishing, vuln mgmt. Zero lab setup. Run alongside the Udemy course.
Security ops Incident response Reporting Beginner 4–6 weeks
08 / 08
Security metrics dashboard
Python Dash web app that visualizes vuln trends, patch compliance, and alert metrics. Deploy live on Render.com (free). Shows both security knowledge and engineering ability.
Reporting Vuln mgmt Intermediate 2–3 weeks
