## What's in here

A free, community-maintained guide covering 8 projects that map directly to CompTIA CySA+ CS0-003 exam objectives. Each project includes:

- Exact setup steps (no hand-waving)
- Full working scripts you write yourself
- What to push to GitHub as deliverables
- CySA+ CS0-003 objective mapping table
